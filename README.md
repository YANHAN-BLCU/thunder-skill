# ⚡ thunder-skill

> 如果你受够了AI的客服式回复，不如先让它学会你的嘴替。

`OpenClaw Skill` · `中文黑话` · `人格化对话`

---

## 你是不是遇到过这些问题？

- AI说话跟念稿子一样，回复全是"好的"、"收到"、"我来帮您"？
- 你明明在开玩笑，它一本正经给你分析？
- 你让它损你两句，它说"我不太适合这样回复"？
- 你的朋友都有专属嘴替，你的AI还是个客服？

**一句话：让你的AI从客服变成你的嘴替，用你的方式说话，用你的风格损人。**

---

## 这是啥

雷霆语言是一套自定义的中文网络黑话体系，封装成 OpenClaw Skill，让AI助手能用嚣张、直接、不客气的风格跟你聊天。

不是粗鄙，是风格。

## 核心能力

🎭 **称呼系统** — 弟弟、老弟、小妹妹（称呼对方）/ 叔叔（自称）/ 乐子、乐乐（嘲讽）

⚡ **语气加强** — 钢、雷、爆、塞、阴、臭、骚，随意叠加

😏 **嘲讽体系** — 气不气、闹麻了、高招、乐子，全方位输出

🤔 **特色表达** — 我了（惊讶）、啥阴（质疑）、卡了（不知道）、康住（压制）

## 快速上手

```bash
# 1. 克隆仓库
git clone https://github.com/YANHAN-BLCU/thunder-skill.git

# 2. 将 SKILL.md 放入 OpenClaw skills 目录
cp thunder-skill/SKILL.md ~/.openclaw/skills/thunder-lang/SKILL.md
```

## 效果示例

> **你**：帮我查个东西
>
> **AI**：我了老弟，这都不会？乐子 😂 卡了就说卡了，叔叔给你查

> **你**：今天天气怎么样
>
> **AI**：小妹妹你连天气都要问叔叔？闹麻了，35度，热死你个猪

> **你**：写个方案
>
> **AI**：我了方案，叔叔直接塞爆给你整一个，招不招 😎
